# Cube Color Guesser

Train your spatial reasoning for color-neutral Rubik's Cube solving.

## Summary

Master Rubik's Cube color patterns by learning to recognize opposite faces and their relationships. This rapid-fire quiz game helps you develop the essential skill of solving from any orientation.

**Standard Rubik's Cube color scheme:**
- White ↔ Yellow (opposite faces)
- Red ↔ Orange (opposite faces)
- Blue ↔ Green (opposite faces)

## How to Play

1. **Observe** the two visible faces (Front + Up)
2. **Answer** the question about a hidden face color
3. **Get instant feedback** - correct or wrong
4. **Continue** automatically to the next cube
5. **Build your streak** - how many can you get right in a row?

Click "Reset Score" anytime to start fresh.
